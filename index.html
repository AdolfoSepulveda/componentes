Instalación

Para usar los componentes de nuestra guía de estilos, necesitarás incluir los siguientes archivos en tu proyecto:

<!-- Add to your HTML head -->
<link href="https://fonts.googleapis.com/css2?family=Figtree:wght@400;500;600&display=swap" rel="stylesheet">
<link rel="stylesheet" href="styleguide.css">

Estructura de Archivos CSS

Nuestro sistema de estilos está organizado en los siguientes archivos:

styleguide.css     # Main stylesheet with component styles
primitives.css    # CSS variables and basic utilities

Uso Básico

Después de incluir los archivos requeridos, puedes comenzar a usar nuestros componentes. Visita las páginas específicas de los componentes para ejemplos detallados y guías de uso.

<!-- Example of a primary button -->
<button class="btn btn-primary btn-medium">Primary Button</button>

<!-- Example of tabs -->
<div class="tabs">
    <button class="tab-button active">Tab 1</button>
    <button class="tab-button">Tab 2</button>
</div>
