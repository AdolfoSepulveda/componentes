Comenzar

Requisitos Previos

Antes de comenzar, asegúrate de tener:

  • Un editor de código
  • Acceso a los archivos CSS del sistema de diseño
  • Conocimientos básicos de HTML y CSS

Instalación

Para usar los componentes de nuestra guía de estilos, sigue estos pasos:

1. Descarga los Archivos

Descarga los siguientes archivos CSS esenciales:

  • styleguide.css - Estilos principales de los componentes
  • primitives.css - Variables CSS y utilidades básicas

2. Incluye los Archivos en tu Proyecto

Agrega las siguientes líneas en el <head> de tu HTML:

<!-- Add to your HTML head -->
<link href="https://fonts.googleapis.com/css2?family=Figtree:wght@400;500;600&display=swap" rel="stylesheet">
<link rel="stylesheet" href="primitives.css">
<link rel="stylesheet" href="styleguide.css">

Estructura de Archivos CSS

Nuestro sistema de estilos está organizado en los siguientes archivos:

styleguide.css     # Main stylesheet with component styles
primitives.css    # CSS variables and basic utilities
  ├─ Colors      # Color variables and themes
  ├─ Typography  # Font sizes, weights, and families
  ├─ Spacing     # Margin and padding utilities
  └─ Radius      # Border radius utilities

Uso Básico

Después de incluir los archivos requeridos, puedes comenzar a usar nuestros componentes. Aquí hay algunos ejemplos básicos:

Botones

<!-- Botón Primario -->
<button class="btn btn-primary btn-medium">Primary Button</button>

<!-- Botón Secundario -->
<button class="btn btn-secondary btn-medium">Secondary Button</button>

<!-- Botón de Peligro -->
<button class="btn btn-danger btn-medium">Danger Button</button>

Pestañas

<!-- Ejemplo de Pestañas -->
<div class="tabs">
    <button class="tab-button active">Tab 1</button>
    <button class="tab-button">Tab 2</button>
    <button class="tab-button">Tab 3</button>
</div>

Campos de Entrada

<!-- Campo de Texto -->
<input type="text" class="input input-medium" placeholder="Enter text">

<!-- Campo con Etiqueta -->
<div class="input-group">
    <label class="input-label">Username</label>
    <input type="text" class="input input-medium">
</div>
